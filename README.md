# 🌍 Cesium Contamination Prediction using Machine Learning  
### Fukushima Nuclear Accident Dataset

## 📌 Overview
This project predicts Cesium contamination levels using Machine Learning techniques based on environmental data collected after the Fukushima Nuclear Accident.

## 🚀 Objectives
- Data preprocessing and cleaning
- Model training and evaluation
- Prediction of contamination levels
- Performance comparison of ML models

## 🧠 Machine Learning Models
- Linear Regression
- Random Forest
- Decision Tree
